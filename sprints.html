Mémoire politique

Le site internet Mémoire politique est un outil développé par La Quadrature du Net pour permettre aux citoyens de l'Union européenne de contacter leurs députés, et de retrouver leurs positions lors des votes concernant les libertés sur Internet.

Site :

  • http://www.laquadrature.net/wiki/Memopol
  • https://memopol.lqdn.fr/

Contact : tb at laquadrature dot net / contact at laquadrature dot net

Circus

Circus est un superviseur de process écrit en python, vous pouvez l'utiliser pour contreoller et superviser vos process et vos sockets. Il est similaire en fonctionalités à Supervisord.

http://circus.readthedocs.org/en/0.9.2/

Une partie de l'équipe derrière Circus sera présente à Pycon et nous en profiteront pour sprinter sur certaines taches (support de python 3, passage des commandes en asynchrone, amélioration de la documentation), venez nous parler !

Contact: alexis [chez] mozilla [point] com

Pypy

PyPy est une implémentation de Python axée sur les performances et le respect des spécifications, le sprint n'a pas d'objectif particulier mais voici des exemples de taches qui pourront être réalisées :

  • Faire fonctionner sur PyPy le code que les participants apporteront
  • Améliorer le support de Numpy
  • Résoudre des bugs prévu pour les débutants

Il n'est pas nécessaire d'avoir précédemment contribué à PyPy pour participer mais assister à la présentation du dimanche est conseillé.

Zim

Description : Zim est un wiki de bureau écrit en Python, à l'aide de GTK+. Il permet la prise et l'organisation de notes au format wiki ; synthèse, liste de tâches, retour d'expérience : stocker son savoir et y accéder facilement. De nombreuses fonctionnalités complémentaires sont disponibles en plugins.

Site : http://zim-wiki.org/

Tâches : http://www.zim-wiki.org/wiki/doku.php?id=development_planning

Scikit-learn

Description : scikit-learn est un module Python intégrant des algorithmes d'apprentissage statistiques classiques dans l'univers de Python scientifique (numpy, scipy, matplotlib). Il offre des solutions simples et efficaces pour résoudre des problèmes d'apprentissage, accessible à tous, utilisable dans des domaines variés.

Site : http://scikit-learn.org/

Daybed

Description: Daybed est un service HTTP de validation de modèles: vous définissez un modèle et les données seront par la suite vérifiées avant d'être insérées en base de donnée.

Les applications sont multiples puisque il s'agit d'une opération que l'on effectue sur la plupart de nos sites.

L'objectif du sprint est de travailler sur l'interface utilisateur et la documentation, puisque le projet est quasiment complet.

Site: https://github.com/spiral-project/daybed

Contact: spiral-project chez lolnet.org
